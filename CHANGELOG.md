# Changelog

## v1.0.0

Initial public release.

### Added

- Reduce squad acknowledgement voice volume.
- Keep important combat and alert transmissions unchanged.
- Config file generated through BepInEx:
  - `ifz.squadvoicevolume.cfg`
- Separate volume values for:
  - squad selection
  - movement confirmations
  - attack acknowledgements
  - building clear confirmations
- Lightweight direct hook on the game radio voice player.
